# Smart Kitchen Management System

## Overview

The Smart Kitchen Management System is a UX/UI design project that helps users discover recipes, search by ingredients, apply filters, and follow guided cooking instructions through an intuitive mobile interface.

## Features

- Search recipes by name or ingredient
- Filter recipes by user preferences
- View detailed recipe information
- Guided cooking experience
- Mobile-friendly interface

## My Contribution

I designed the Cooking Help feature by creating user flows, storyboards, hand-drawn wireframes, and digital wireframes while applying user-centered design principles.
